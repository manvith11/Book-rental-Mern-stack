# Book Rental Platform

The Book Rental Platform is a web application developed using the MERN stack (MongoDB, Express.js, React, Node.js). This platform allows users to rent books, manage deliveries, and handle payments seamlessly. 

## Features

- **Book Rental**: Users can browse through a wide range of books and rent them.
- **Delivery Management**: Renters can schedule deliveries and track their status.
- **Payment Processing**: Secure payment integration for handling transactions between renters and book owners.
- **Advanced Search Filters**: Find books efficiently using filters such as genre, author, and profile.
- **User Profiles**: Manage personal profiles and track rental history.

## Technologies Used

- **MongoDB**: NoSQL database for storing user and book data.
- **Express.js**: Web application framework for Node.js to handle server-side logic.
- **React**: Front-end library for building interactive user interfaces.
- **Node.js**: Server-side JavaScript runtime for handling application logic.
